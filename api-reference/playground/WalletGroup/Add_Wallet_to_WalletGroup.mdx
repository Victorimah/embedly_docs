---
title: "Add Wallet to WalletGroup"
api: "POST /walletgroups/{groupId}/wallet/{walletId}/add"
description: "The endpoint allows you to add a wallet to a specified wallet group."
---

## Request
### **Path Parameters**

<ParamField body="groupid" type="string" required>
  The unique identifier for the wallet group.  
</ParamField>

<ParamField body="walletid" type="string" required>
  The id of the wallet to add.
</ParamField>


## Response

✅ Success Response

```json

Status Code: 200 OK

{
  "code": "00",
  "success": true,
  "message": "Wallet added successfully.",
  "data": {
    // Additional details if applicable
  }
}

```
