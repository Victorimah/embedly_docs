---
title: "Customer Limit"
api: "PUT /limits/customer/add"
description: "The endpoint allows you to add limits for a specific customer."
---

## Request Body

<ParamField body="currencyId" type="string" required>
  The unique identifier for the currency (e.g., "USD", "EUR").
</ParamField>

<ParamField body="productId" type="string" required>
  The unique identifier for the product associated with the limit.
</ParamField>

<ParamField body="customerId" type="string" required>
  The unique identifier for the customer to whom the limit applies.
</ParamField>

<ParamField body="singleTransactionLimit" type="number" required>
  The maximum amount allowed for a single transaction.
</ParamField>

<ParamField body="frequency" type="number" required>
  The number of transactions allowed within the specified period.
</ParamField>

<ParamField body="dailyTransactionLimit" type="number" required>
  The maximum total amount allowed for transactions in a single day.
</ParamField>

<ParamField body="monthlyTransactionLimit" type="number" required>
  The maximum total amount allowed for transactions in a single month.
</ParamField>

<ParamField body="id" type="string" required>
  The unique identifier for the limit being added.
</ParamField>
