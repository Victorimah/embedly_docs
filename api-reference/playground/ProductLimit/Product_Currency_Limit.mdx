The `GET /api/v1/limits/product/{productId}/currency/{currencyId}` endpoint allows you to retrieve limits for a specific product and currency.

### Request Method

`GET`

### **Path Parameters**

- `productId`: The ID of the product (e.g., `769dca4b-7d42-43b5-ade6-bcc10d611f6f`).
- `currencyId`: The ID of the currency (e.g., `555`).

---

## Response
✅ Success Response
```json
Status Code: 200 OK

{
  "code": "00",
  "success": true,
  "message": "Retrieved successfully",
  "data": {
    "id": "769dca4b-7d42-43b5-ade6-bcc10d611f6f",
    "productCurrencyId": "555",
    "singleTransactionLimit": 10000000,
    "frequency": 10000,
    "dailyTransactionLimit": 500000000,
    "monthlyTransactionLimit": 500000000,
    "customerId": null
  }
}
```


## ❌ Error Responses
```json
- **Status Code**: **400 Bad Request**

{
  "code": "-9",
  "success": false,
  "message": "Invalid currencyId specified",
  "data": null
}
