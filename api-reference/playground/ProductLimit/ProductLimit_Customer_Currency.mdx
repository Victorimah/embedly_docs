---
title: "Limit for Customer"
api: "GET /limits/product/{productId}/currency/{currencyId}/customer/{customerId}"
description: "The endpoint allows you to retrieve limits for a specific product and currency for a given customer."
---

### **Path Parameters**

- `productId`: The ID of the product (e.g., `a529a0c8-ff67-11ef-a8a9-6045bd97b81d`).
- `currencyId`: The ID of the currency (e.g., `a529a0c8-ff67-11ef-a8a9-6045bd97b81d`).
- `customerId`: The ID of the customer (e.g., `08dd5d6c-a9b4-452d-8872-4ada3d46b506`).
