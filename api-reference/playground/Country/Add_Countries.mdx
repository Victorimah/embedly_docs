---
title: "Add Countries"
api: "POST /Countries/add"
description: "This endpoint enables you to add a Country"
---

## Request Body

<ParamField body="id" type="string">
  Unique identifier for the country.
  **Example:** "3fa85f64-5717-4562-b3fc-2c963f66afa6"
</ParamField>

<ParamField body="name" type="string" required>
  Full name of the country.
  **Example:** "Ghana"
</ParamField>

<ParamField body="countryCodeTwo" type="string" required>
  Two-letter ISO country code (ISO 3166-1 alpha-2).
  **Example:** "GH"
</ParamField>

<ParamField body="countryCodeThree" type="string" required>
  Three-letter ISO country code (ISO 3166-1 alpha-3).
  **Example:** "GHA"
</ParamField>


# Response

##✅ Success Response

- **Status Code**: `200 OK`

```json
{
  "code": "00",
  "success": true,
  "message": "Country Added successfully",
  "data": {
    "id": "08dd7754-454d-4ab0-8ed1-5eed930ad067",
    "name": "Ghana",
    "countryCodeTwo": "GH",
    "countryCodeThree": "GHA"
  }
}
```
