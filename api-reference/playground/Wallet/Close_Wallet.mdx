---
title: "Close Wallet"
api: "POST api/v1/wallets/close"
description: "The endpoint allows you to close a specified wallet using the account number and associated details."
---

### **Request Body**

```json
{
  "accountNumber": "3023563113",
  "accountClosureReasonId": 0,
  "tellerId": 0,
  "closeOrDelete": true,
  "customerOrAccount": true
}
```

## Response

```json
✅ Success Response
Status Code: 200 OK

{
  "code": "200",
  "success": true,
  "message": "Wallet closed successfully.",
  "data": {
    // Additional data fields if applicable
  }
}

```
