The `PATCH /api/v1/products/activate/{productId}` endpoint allows you to activate a product by its ID.


## Request Method
`PATCH`

### **Path Parameter**
- `productId`: The ID of the product to activate (e.g., `1404e81a-0bf1-11f0-a8a9-6045bd97b81d`).
---

Status Code: 200 OK

  }
