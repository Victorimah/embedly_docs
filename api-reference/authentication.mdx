<Info>
	The APIs on the staging environment are currently open and do not require an
	API key for authentication. However, they are subject to change with prior
	notice.
</Info>

## Base URL

The base URL for all API requests on the production environment is:

```
https://api.embedly.co/v1
```

While the base URL for all API requests on the staging environment is:

```
https://waas-staging.cafeone.ng/api/v1
```

## Authentication

You are to pass your API-KEY as a header in all your requests on the platform.

**Headers**

```https
x-api-key: {{API-KEY}}
```
